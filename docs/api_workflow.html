post_job(pipeline_name, project_name, params, script, revision, build_source, build_context, profile=None, trace=False)[source]¶

Run a workflow job

Parameters

  • pipeline_name (Optional[str]) – Name of the pipeline to run

  • project_name (str) – Name of the project to run this job on

  • params (Optional[List]) – List of parameters to forward to the job

  • script (Optional[str]) – Git repository url to run (only when server is in development mode)

  • revision (Optional[str]) – Git revision or tag (only when server is in development mode)

  • build_source (Optional[str]) – Source type of the nextflow build (builtin, git, url)

  • build_context (Optional[str]) – Context for build_source, depends on the type.

  • profile (Optional[str]) – Nextflow profile name to use.

  • trace (bool) – Instruct the job to run nextflow trace flags for debugging.
